## 关于瑞克商业说

我是瑞克，一名从阿里中供铁军和字节飞书走出来的商业咨询顾问。过去 10 年，我经历了国企军工、欧司朗精益生产、阿里 SaaS 区域负责人、字节飞书高级商业化总监，如今专注于帮助中小商家解决三个核心问题：

- **获客**：怎么获得更多客户？
- **团队**：刺头员工怎么管？销售团队怎么搭？
- **效率**：没有流程的勤奋是在给漏洞打补丁

我的信条：低级的商业解决痛点，高级的商业销售梦想。但梦想的前提是，先把痛点解决得足够扎实。

## 这个仓库是什么

Agent Skills 是 AI 智能体的可复用技能包。每个 skill 是一个文件夹，包含一份 `SKILL.md` 指令文件，告诉 AI 如何在一个具体场景中完成具体任务。

这里的每一个 skill 都经过我的实战验证——不是纸上谈兵，是我每天工作中真正在用的工具。

## 使用方式

⚠️ **重要提示：当前技能为 Tabit 浏览器 Agent 专用格式**

这些技能使用 Tabit 浏览器独有的自动化引擎（点击、弹窗处理、下载管理、页面元素定位），**只能直接在 Tabit 浏览器中执行**。在其他平台（如 Kimi Work、Claude Code、Codex CLI 等）只能作为 SOP 操作手册阅读，无法自动执行 GUI 操作步骤。

**Tabit 用户使用方式：**
1. 在 Tabit 浏览器中打开本仓库链接
2. 将 `skills/` 目录下的技能文件夹添加到 Tabit 的 Agent 技能库
3. 在对话中直接发送指令，如："执行今日灰豚数据导出"

**非 Tabit 用户：**
- 可作为标准操作流程（SOP）参考，手动按步骤执行
- 或等待后续发布的 Python + Playwright 通用脚本版本
