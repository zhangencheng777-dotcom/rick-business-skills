## 关于瑞克商业说

我是瑞克，从阿里中供铁军和字节飞书走出来的商业咨询顾问。过去 10 年经历了国企军工、艾欧史密斯（A.O.史密斯）精益生产、阿里巴巴国际站区域 SaaS 负责人、字节飞书高级商业化总监。如今专注帮中小商家解决获客、团队管理、效率提升三个问题。

低级的商业解决痛点，高级的商业销售梦想。这里的每一个 skill 都经过我实战验证，不是纸上谈兵。

## 这个仓库是什么

Agent Skills 是 AI 智能体的可复用技能包。每个 skill 是一个文件夹，包含一份 `SKILL.md`，告诉 AI 如何在一个具体场景中完成具体任务。

## 使用方式

**tabbit 用户：** 将 `skills/` 下技能文件夹添加到 tabbit Agent 技能库，对话中直接发指令即可执行。

**非 tabbit 用户：** 可作为 SOP 操作手册参考，无法自动执行 GUI 操作。

| 平台 | 能否执行 |
|------|---------|
| **tabbit 浏览器** | ✅ 直接执行 |
| Kimi Work / Claude Code / Codex | ❌ 无法自动执行 |
