## 关于瑞克商业说

我是瑞克，一名从阿里中供铁军和字节飞书走出来的商业咨询顾问。过去 10 年，我经历了国企军工、欧司朗精益生产、阿里 SaaS 区域负责人、字节飞书高级商业化总监，如今专注于帮助中小商家解决三个核心问题：

- **获客**：怎么获得更多客户？
- **团队**：刺头员工怎么管？销售团队怎么搭？
- **效率**：没有流程的勤奋是在给漏洞打补丁

我的信条：低级的商业解决痛点，高级的商业销售梦想。但梦想的前提是，先把痛点解决得足够扎实。

## 这个仓库是什么

A-Skills（Agent Skills）是 AI 智能体的可复用技能包。每个 skill 是一个文件夹，包含一份 `SKILL.md` 指令文件，告诉 AI 如何在一个具体场景中完成具体任务。

这里的每一个 skill 都经过我的实战验证——不是纸上谈兵，是我每天工作中真正在用的工具。

## 使用方式

支持主流 AI 智能体平台：
- **Kimi Work / Kimi Code**：将 `skills/` 目录下的技能文件夹复制到你的本地 skills 目录
- **Claude Code**：安装插件或复制到 `~/.claude/skills/`
- **Codex CLI**：通过 `skills.sh` 安装或直接复制到 `.agents/skills/`

每个 skill 文件夹包含标准 `SKILL.md` 文件，AI 会自动读取并根据场景触发。
